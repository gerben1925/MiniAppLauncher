# Mini App Launcher (Backend)

> Personal backend project built with ASP.NET Core and Clean Architecture, currently under active development.

---

## Status
This project is currently **ongoing and under active development**.

---

## About the Project

This is a **personal project** created to showcase and apply the skills I have developed over the past years in backend development. It focuses on building a scalable, maintainable, and production-style system using modern best practices.

---

## 🚀 Key Features

- Clean Architecture (UseCases, Repositories, Services)
- ASP.NET Core Web API (RESTful APIs)
- Dapper for high-performance data access
- JWT Authentication with Refresh Tokens
- Email Verification & Reset Password workflows
- Role-Based Access Control (RBAC)
- User Management (CRUD, roles, status)
- App Management (CRUD, publish/draft)
- Token Management (verification, reset)
- Email Notification System with reusable templates
- Dependency Injection, Logging, Error Handling

---

- .NET 6 / 8
- ASP.NET Core Web API
